📸 Studio Foto AI

Sebuah aplikasi web interaktif yang dikuasakan oleh AI generatif (Gemini API) untuk mengubah gambar biasa anda menjadi gambar studio profesional, pelbagai tema, dan pelbagai gaya seni secara magis!

Aplikasi ini dibina dengan Antara Muka Pengguna (UI) yang ceria, moden, dan responsif.

✨ Ciri-Ciri Utama

Pelbagai Gaya Seni: Realistik, Pixar 3D, Anime Ghibli, Cat Air, Siberpunk, Pop Art, dan banyak lagi.

Pakaian & Aksesori: Tukar pakaian kepada Kasual, Sut Korporat, Pakaian Tradisional, atau muat naik rujukan pakaian anda sendiri.

Latar Belakang Tersuai: Pilih dari pelbagai tema latar (Studio, Alam Semula Jadi, Bandar Moden) atau muat naik gambar latar anda sendiri.

Kekalkan Identiti: Teknologi AI yang dioptimumkan untuk mengekalkan bentuk wajah dan identiti individu asal dengan ketat.

Sejarah Awan (Cloud History): Gambar yang dijanakan akan disimpan secara automatik ke dalam Firebase Firestore supaya anda tidak kehilangannya.

Kualiti Tinggi (Print-Ready): Menyokong muat turun gambar beresolusi tinggi dengan kualiti gred komersial.

🛠️ Teknologi yang Digunakan

Frontend: React.js, Tailwind CSS

Ikon UI: Lucide React

Backend / Pangkalan Data: Firebase (Authentication & Firestore)

Kecerdasan Buatan (AI): Google Gemini 2.5 Flash Image Preview API

🚀 Cara Menjalankan Projek (Local Development)

Clone repositori ini:

git clone [https://github.com/username-anda/studio-foto-ai.git](https://github.com/username-anda/studio-foto-ai.git)


Masuk ke dalam direktori projek:

cd studio-foto-ai


Pasang semua pakej yang diperlukan:

npm install firebase lucide-react


Tetapkan API Key Gemini anda di dalam fail .env:

VITE_GEMINI_API_KEY=API_KEY_ANDA_DI_SINI


Jalankan pelayan pembangunan (development server):

npm run dev


📦 Deployment

Aplikasi ini sedia untuk dilancarkan (deploy) menggunakan platform popular seperti Vercel, Netlify, atau Firebase Hosting. Pastikan anda menetapkan Environment Variables untuk API Key Gemini anda di platform tersebut sebelum melancarkannya.
